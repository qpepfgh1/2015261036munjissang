# GitHub Markdown 문법 정리  2015261036 문지상

### 1. 헤더(Headers)
<br>
#### 큰 제목 : 문서 제목<br>

사용 방법은 아래와같이 코딩하면 된다.
<pre>
  <code>
  문서 제목
  ======== 
  </code>
</pre>

#### 실행 결과

문서 제목
========
<br>
<br>
<br>
#### 작은 제목 : 문서 부 제목<br>

사용 방법은 아래와같이 코딩하면 된다.
<pre>
  <code>
  문서 부 제목
  ----------- 
  </code>
</pre>

문서 부 제목
-----------
